## 项目说明

本仓库包含本人在南京大学 DLCO 课程中的部分实验代码与电路实现。

由于曾经手滑点到了实验平台的 **“重置代码”** 按钮，  
导致 Lab1 和 Lab2 的代码直接蒸发，所以这里友情提醒：

> 在点击任何危险按钮前，请务必备份代码！

## 关于这些电路

一开始其实没太学明白这门课  
（骗你的，其实现在也没完全学明白 QAQ）

因此从 **Lab4** 开始，才逐渐加入自己的思考题实现与设计。

从那之后，我的电路风格也开始逐渐走向：

- 顶级隧道（Tunnel）使用者
- ROM 狂热爱好者
- 能省线绝不多拉一根派

所以部分实现可能会比较“狂野”。

---

## 使用建议

如果你还没有完全理解实验原理：

> **不要直接照抄电路。**

否则可能会出现：

- 验收时被老师现场追问
- 出 bug 后完全不会调试
- 看懂自己的电路比重新写还困难

建议把这些实现当作：

- 思路参考
- Debug 对照
- 或 Logisim 邪道技巧展示
