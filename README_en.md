## About This Repository

This repository contains part of my lab work and circuit implementations for the DLCO course at Nanjing University.

Unfortunately, I once accidentally clicked the **“Reset Code”** button on the lab platform,  
which instantly erased all my Lab1 and Lab2 files.

So here is a sincere reminder:

> ⚠️ Always back up your work before clicking suspicious buttons.

## About the Circuits

At the beginning, I honestly didn’t really understand this course very well.

(Actually... I still don’t fully understand it now.)

Because of that, my own designs and solutions only really started appearing from **Lab4** onward.

And from that point on, my circuit style gradually evolved into:

- Extreme Tunnel enjoyer
- ROM enthusiast
- “One less wire is one more victory” believer

So some implementations may look a little... unconventional.

---

## Recommendation

If you haven't fully understood the principles behind the labs yet:

> **Please don’t blindly copy the circuits.**

Otherwise you may experience:

- Getting questioned during check-offs
- Random unexplained bugs
- Circuits exploding mysteriously
- Spending more time debugging than rebuilding

It’s better to use this repository as:

- A reference for ideas
- A debugging comparison
- Or a collection of questionable Logisim engineering techniques
